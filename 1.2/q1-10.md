```scheme
(define (A x y)
  (cond ((= y 0) 0)
        ((= x 0) (* 2 y))
        ((= y 1) 2)
        (else (A (- x 1)
                 (A x (- y 1))))))
```

アッカーマン関数に関しての詳細は、以下を参照。

* https://ja.wikipedia.org/wiki/アッカーマン関数

で、問題にある処理を実行した結果は以下となる。

```scheme
(A 1 10)
;; 1024

(A 2 4)
;; 65536

(A 3 3)
;; 65536
```

---

## (f n)

* 置き換えモデル
```scheme
(define (f n) (A 0 n))
-> (* 2 n)
```

* 実際の処理結果
```schema
(f 0)
;; 0

(f 1)
;; 2

(f 2)
;; 4

(f 3)
;; 6

(f 4)
;; 8
```

```
f(n) = 2n
```

## (g n)

* 置き換えモデル
```scheme
(define (g n) (A 1 n))
-> (A 0 (A 1 (- n 1)))
-> (A 0 (A 1 (- n 1)))             ;; (* 2 (A 1 (- n 1)))
-> (A 0 (A 0 (A 1 (- n 2))))       ;; (* 2 (* 2 (A 1 (- n 2))))
-> (A 0 (A 0 (A 0 (A 1 (- n 3))))) ;; (* 2 (* 2 (* 2 (A 1 (- n 3)))))
-> (A 0 (A 0 (A 0 ... (A 1 1))))   ;; (* 2 (* 2 (* 2 ... (* 2 2))))
```

* 実際の処理結果
```scheme
(g 0)
;; 0

(g 1)
;; 2

(g 2)
;; 4 -> 2^2

(g 3)
;; 8 -> 2^3

(g 4)
;; 16 -> 2^4
```

```
g(n) = 2^n
g(0) = 0
```

## (h n)

* 置き換えモデル
```scheme
(define (h n) (A 2 n))
-> (A 1 (A 2 (- n 1)))             ;; 2^(A 2 (- n 1))
-> (A 1 (A 1 (A 2 (- n 2))))       ;; 2^2^(A 2 (- n 2))
-> (A 1 (A 1 (A 1 (A 2 (- n 3))))) ;; 2^2^2^(A 2 (- n 3))
-> (A 1 (A 1 (A 1 ... (A 2 1))))   ;; 2^2^2^...^2
```

* 実際の処理結果
```scheme
(h 0)
;; 0

(h 1)
;; 2

(h 2)
;; 4 -> 2^2

(h 3)
;; 16 -> 2^4

(h 4)
;; 66536 -> 2^16
```


```
h(n) = 2^h(n-1) ... n > 1 の時
h(1) = 2
h(0) = 0
```
